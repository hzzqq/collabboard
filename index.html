房主：—
多人打开同一房间即可实时同步（含光标）。服务端 server.js 用 Node 原生 net 手写 WebSocket，按房间隔离广播。

右侧聊天栏可与同房间伙伴对话；左栏可设置昵称、查看在线名单。

启动：node server.js
快捷键：Ctrl+Z 撤销 / Ctrl+Y 重做
「选择/移动」工具可拖动已画元素（文字/图形整体平移，松手同步他人）。▢ 框选工具拖出矩形即可多选，选中后拖动任一元素会整组移动（服务端按 id 列表平移并广播）；☑ 全选/✕ 取消选择或 Esc 管理选区。
